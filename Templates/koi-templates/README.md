# KOI Templates
This folder contains templates for KOI-Sync notes.

## Setup

To use these templates with KOI-Sync:
1. Enable KOI-Sync plugin
2. Settings → KOI-Sync → Template file location: `Templates/koi-templates`
3. Settings → KOI-Sync → Sync directory: `Inputs/koi`
4. Add your KOI API credentials

Your KOI notes will sync to `Inputs/koi` (which is gitignored and stays private).
