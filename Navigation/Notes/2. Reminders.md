## Notes
```dataview
TASK
FROM "Notes"
WHERE !completed  AND class = "Notes"
GROUP BY file.link
```

## Reference Notes
```dataview
TASK
WHERE !completed AND class = "Reference_Notes"
GROUP BY file.link
```
